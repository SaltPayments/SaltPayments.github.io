Try It Yourself

You can use this page to make a test transaction. This will allow you to observe the functionality of the SALT platform before signing up for a sandbox account.
  • Click Submit on the form below to create a test transaction. The transaction response will be displayed.
  • Log in to http:// with username/password: username/password
  • Click on Payments in the left-hand navigation bar
  • You will be able to view a list of all recent transactions. Clicking on any of these transaction will provide further details.
