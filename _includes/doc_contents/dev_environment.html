2 The SALT Development Environment

The SALT Development Environment is a sandbox environment mirroring the production environment that allows you to perform testing and integration before going into production.

Once you have a merchant account setup with SALT, SALT's administrator will create your Credit Card Core API Stores for you and grant you access to the Development environment.


Table 2-1 - SALT Core API Development Environment

URL

Description

SALT Merchant Management Website

https://test.salt.com/merchantmanagement/

Allows you to perform administrative tasks such as viewing transaction activity and configuring your Stores.

Credit Card Core API Handler

https://test.salt.com/gateway/creditcard/processor.do

Receives Credit Card Core API requests

The SALT Merchant Management Website requires a login, which consists of a Username and Password provided to you upon signup. You can change your password after logging in.


2.1 Store Information and Configuration

To access the Store information/configuration page:

1. Login to the SALT Merchant Management Website at https://test.salt.com/merchantmanagement/

2. Select the Store Link under the Merchant Setup section. A list of all of your Stores will be displayed.

3. Press the Modify button for the Store that you wish to configure. You will be taken to the Store configuration page.

Information about your Store (including Merchant ID and Store ID) is shown, along with all configuration options.
