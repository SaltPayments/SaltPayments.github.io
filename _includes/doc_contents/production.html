8 Moving to Production

Once certification is complete, you will be ready to move to the SALT Production environment. Moving to the Production environment involves changing the credit card Core API handler URL to the production URL, and verifying your configuration settings in the production Merchant Management Website.


Table 8-1 SALT Core API Production Environment

URL

SALT Merchant Management Website

https://manage.salt.com/

Core API Handler

https://gateway.salt.com/gateway/creditcard/processor.do

When you want to move to production:

1. Notify SALT so that we can enable your account on our production system.

2. Where you invoke the SALT Core API, change the SALT Credit Card Core API URL fromhttps://test.salt.com/gateway/creditcard/processor.do to https://gateway.salt.com/gateway/creditcard/processor.do. See Step 1 of Figure 1-1.

3. Finally, verify that your account is enabled and that your configurations are correct by logging into the production Merchant Management Website at https://manage.salt.com/.
