Refund

Returns funds from a previously settled purchase to the customer. Refunds can only be performed on a purchase that is part of an already-closed batch.


IOPHttpsTransaction.refund Arguments Description
int merchantIdThe ID of the merchant issuing the requests.
String apiTokenThe API token of the merchant issuing the requests.
String storeIdThe merchant's store ID.
long transactionIdThe transaction ID returned when the original purchase was made.
long amountAmount of the purchase in cents.
