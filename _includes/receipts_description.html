CreditCardReceipt getCvv2Response() Returns a Cvv2Response when performing a transaction with a VerificationRequest. getPeriodicPurchaseInfo() Returns PeriodicPurchaseInfo when performing a recurring payment. getStorageTokenId() Returns the storage token used when setting up Secure Storage. getOrderId() Returns the merchant assigned orderId for a purchase or refund. getResponse() Returns the response from the creditcard gateway.
