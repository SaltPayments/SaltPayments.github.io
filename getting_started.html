Getting Started

  • Java
    1. Create Test Merchant Account. Generate API token, merchant Id, user name and password.
      1. Contact Customer Services
        • Phone: 1-866-854-9851 ext. 2
        • Email: support@salt.com
    2. Get source code as zip file or from git repository
      • Zip File
        1. Download the repository zip file
        2. Extract the zip file
      • Git
        1. Check out the from git hub repository
    3. Update API token and merchant Id in source code
      1. Open merchant.properties file located under salt-payment-client-java\src\main\resources\merchant.properties
      2. Replace the values of merchant.Id and merchant.apiToken with those received from step one
    4. Make a transaction
      • Maven:
        1. To run example single purchase execute: mvn install -Psingle-purchase
        2. To run example secure storage execute: mvn install -Psecure-storagee
        3. To run example recurring-purchase execute: mvn install -Precurring-purchase
      • Intellij:
        1. Open a new project in Intellij by opening the pom.xml file, extracted from the zip file.
        2. Open the maven tab (located at the left side).
        3. Run the lifecycle named "clean" followed by the lifecycle named "install".
      • Eclipse:
        1. Run the project as "Maven clean"
        2. Run the project as "Maven install"
    5. View Transaction
      1. Go to the Merchant Management login
      2. Login using the following username and password credentials created in step one
      3. Go to '...' to view transaction
